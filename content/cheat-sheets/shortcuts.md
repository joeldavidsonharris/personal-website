## Snowflake

### UI management

- `Ctrl` + `Alt` + `Up`: Move bottom pane up  
- `Ctrl` + `Alt` + `Down`: Move bottom pane down  
- `Ctrl` + `Alt` + `Left`: Show results in bottom pane  
- `Ctrl` + `Alt` + `Right`: Show chart in bottom pane  

### Code execution

- `Ctrl` + `Enter`: Run query  
- `Ctrl` + `Shift` + `Enter`: Run all queries  

### Code searching

- `Ctrl` + `F`: Find  
- `Ctrl` + `Shift` + `H`: Find and replace  

### Code formatting

- `Ctrl` + `]`: Indent all selected lines  
- `Ctrl` + `[`: Unindent all selected lines  
- `Ctrl` + `/`: Comment all selected lines  

### Cursor movement

- `Up`: Move cursor up  
- `Down`: Move cursor down  
- `Ctrl` + `Left`: Move cursor to start of word  
- `Ctrl` + `Right`: Move cursor to end of word  

### Code editing

- `Alt` + `Up`: Move selected lines up  
- `Alt` + `Down`: Move selected lines down  
- `Ctrl` + `Left Click`: Add cursor  
- `Alt` + `Hold Left Click` + `Drag Mouse`: Add cursors to multiple lines

## dbt Cloud

Use `Fn` + `F1` to show all shortcuts  
Use `__` (double underscore) to insert Jinja templates easily  

### Code execution

- `Ctrl` + `Enter`: Execute and preview 500 rows  
- `Ctrl` + `Shift` + `Enter`: Compile Jinja into SQL  

### Code searching

- `Ctrl` + `F`: Find (expand modal to replace)  

### Code formatting

- `Ctrl` + `]`: Indent all selected lines  
- `Ctrl` + `[`: Unindent all selected lines  
- `Ctrl` + `/`: Comment all selected lines  

### Cursor movement

- `Up`: Move cursor up  
- `Down`: Move cursor down  
- `Ctrl` + `Left`: Move cursor to start of word  
- `Ctrl` + `Right`: Move cursor to end of word  
- `Ctrl` + `Alt` + `Up`: Add cursor above  
- `Ctrl` + `Alt` + `Down`: Add cursor below  

### Code editing

- `Alt` + `Up`: Move selected lines up  
- `Alt` + `Down`: Move selected lines down  
- `Alt` + `Left Click`: Add cursor  
- `Shift` + `Alt` + `Hold Left Click` + `Drag Mouse`: Add cursors to multiple lines
